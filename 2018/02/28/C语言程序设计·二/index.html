发表于 2018-02-28 | 分类于 C/C++

二维数组

T a[n][m]

  • MxN个元素是在内存里一个挨着一个存放的
  • 数组占用了一片连续的，大小为MxNxsizeof(T)字节的存储空间
  • 每一行有M个元素
  • 如果a[0][0]地址为n，那么a[i][j]地址为n+ixm+j
  • 二维数组每一行，实际上是一个一维数组，a[0],a[1]可以看作是一维数组的名字，可以直接使用

函数

  • 调用函数的语句之前用函数声明即可，不用定义
  • 形参是实参的拷贝，形参的改变不会影响实参(数组，指针，对象除外)
  • 数组名做参数(int a[])，不用写个数
  • 数组的长度可以是sizeof(a)/sizeof(int)
  • 二维数组做参数必须要写多少列，不用写明多少行，这样才能根据下标算出地址
    (int a[][5])

库函数

cmath

  • int abs(int x) 绝对值
  • double sqrt(double x) 平方根

ctype

  • int isdigit(int c) 判断c是不是数字字符
  • int isalpha(int c) 判断c是否是一个字母

递归

一个函数自己调用自己，就是递归
就是假设这个函数已经写好了，能够拿到返回值了，在内部直接使用即可

  • 阶乘

  • 求斐波那契数列第n项

位运算

与某个数的第几位相关的运算，都可以考虑位运算
写一个表达式使得该表达式的值和a的第n位相同(a>>n-1)&1

  • &
    两个二进制位为1是才是1
    21&18=16
    用来置0
  • |
    有一个是1就是1
    21|18=23
    用来置1
  • >>右移
    负数高位补1，正数高位补0
    右移一位相当于除以2，并往小里取整
    -25>>4=-2
    -2>>4=-1
    -15>>3=-2
  • <<左移
    高位丢弃，低位补0
    左移1位就是乘以2，而且左移操作比乘法快得多

  • ^异或
    两位不同时为1，相同为0
    21^18=7
    用来按位取反 1111 1111
    如果a^b=c，那么c^b=a,c^a=b此规律能用来进行简单的加密解密
    异或能不用临时变量就交换两个变量的值

  • ~取反

字符串

  • 括号括起来的字符串常量，如”China”
  • 放在字符数组char[]中，以’\0’（是一个字符，ASCII为0）结尾，元素个数应该为字符串长度+1
  • 让一个串中间=0，相当于\0了
  • char数组中每个字符的值，可以随意改变(比JavaString好)
  • string对象/C++
  • 字符串常量占据内存的字节数是字符长度+1，因为/0
  • 字符串长度不包括/0
  • “”也是合法常量，内存空间会占一个字节
  • 双引号 \” 斜杠\

读入字符串

  • cin/scanf 读入对象会在末尾自动加\0
  • scanf/cin读入到空格为止
  • cin.getline 读一行

字符串库函数

  • cstring
  • 形参是char[]，传入的参数可以是常量和char[]
  • strcpy(dest,src) 给字符数组赋值
  • 字符数组不能直接用 “=” 赋值，字符指针可以，而且只有const指针才没有warning
  • strcmp(s1,s2) 字典序
  • strlen
  • strcat
  • strupr
  • strlwr
  • strstr 找子串
  • strtok 找出被指定字符分割的串 要连续调用 坑很多

  • strtok在切割字符串的时间，实际上就是将分割符的字符delimiter替换为’\0’并且返回首地址。

  • atoi 字符转数字

  • itoa 数字转字符

指针

基本概念

  • 每个变量都存在从某个内存地址开始的若干个字节中
  • 指针，也称指针变量，大小为4/8个字节，其内容是一个内存地址
  • 通过指针，能够对指针指向的内存区域进行读写

定义

  • int* p
  • char* pc
  • float* pf

指针的内容

  • int p = (int )4000; p的值是4000
  • 通过p可以修改地址4000里面的内容
  • 星p就是地址4000里面的内容，可以直接赋值

用法

  • 指针存的是地址
  • char ch1 = ‘a’;
  • char* pc = & ch1;
    • pc = ‘b’
  • 不同类型的指针，要通过强制类型转换才能赋值

指针的运算

  • 同类型指针可以比较大小
  • 同类型指针，可以相减
  • p1-p2=(p1地址-p2地址)/sizeof(T)
  • p+n=地址p+ nxsizeof(T)
  • 指针可以用[]来运算，p[n] = *(p+n)
  • 地址0不能访问，指向地址0是空指针
  • NULL就是0

指针作为函数参数

  • 指针做形参可以修改他们的指向的地址的内容，如Swap

数组和指针

  • 数组的名字就是一个指针常量，指向数组开始地址
  • memset() 将n个字符设为统一值

指针和二维数组

  • T a[M][N]
  • a[i]是一个一维数组，且sizeof(a[i]) = sizeof(T) * N
  • a[i]类型是T*
  • a的类型是T( * )[]指向数组的指针

指向指针的指针

  • T** P
    • p 的类型是 T*

指针和字符串常量

  • 字符串常量和字符数组，都是char*
  • 可以直接cout char*

void指针

  • 可以用任何类型的指针对void指针进行初始化或赋值

  • sizeof(void) 无定义，所以*p，p++什么的都无定义

内存操作库函数

mem = memory
参数都是void* 方便得一pi
其实都是一个个逐字赋值的

  • memset
  • memcpy

函数指针

  • 函数也会占用一段连续的内存空间
  • 内存区域开始地址，就是函数名，函数名就是一个指针
  • 将函数的入口地址赋给指针，就能通过指针调用这个函数
  • int (* pf)(int,char)
  • 返回值 （* 指针变量名）（参数类型）
  • 作用 当函数作为参数的时候，可以使用指针 比如qsort
