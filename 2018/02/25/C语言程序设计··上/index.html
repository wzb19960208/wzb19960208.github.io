信息在计算机中的表示

计算机电路由逻辑门电路组成。一个逻辑门电路可以看成一个开关，每个开关的状态是开or关，即对应1或0。所以计算机使用二进制来表示各种信息。

  • 二进制一位，取值只能为0或1，成为一个比特(bit)
  • 八个二进制位，称为一个字节(byte)
  • 1024字节为1KB，1024KB称为1MB如此类推
  • 用8个连续的0或1来表示字母、数字或符号，就有2的8次方组合即256种，足以表示10个阿拉伯数字和英语种的26个字母以及标点了。此即ASCII码编码方案
  • 图片、视频也可以用0和1表示(像素点)
  • 负数在计算机里面使用补码表示

C/C++语言模板

变量/常量

  • int 32位(4字节)有符号
  • double
  • char
  • #define MAX 100
  • 0x开头 16进制 一位16进制数可以用4位2进制数表示
  • 0开头 8进制

获得字节数可以用sizeof()

输入输出

  • %d int
  • %f float
  • %lf输入 %f输出 double(double精度高，一般使用double)
  • 输出宽度 %nd(空格补全) %0nd(前置0补全)
  • 输出精度(%.nf)

cin/cout更方便，但是速度慢

运算符

算术运算符

  • += 这类运算符执行速度更快
  • 表达式值的类型，以操作数种精度高的为准(double>longlong>int>short>char)

逻辑运算符

  • 短路计算 在表达式的值可以断定时会停止，后面的表达式不执行

类型转换

  • (int)f
  • double(n)

优先级

  • ++ – ！(单目)
  • * / %
  • + -
  • < <= > >= == !=
  • &&
  • ||
  • = += -= *= /= %=

用()就完事了

控制语句

  • if
  • switch(必须是整形int，char)
  • for/While/dowhile
  • break/continue(当前循环)

数组

  • int a[100]
  • 数组元素在内存里一个个挨着存放

  • 数组名a是数组的地址，假设为p，变量a[i]地址就是p+i

  • 下标是负值或者太大都会数组越界，但是不会有编译错误
